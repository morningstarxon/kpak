Winning Strategies

Fundamental Principles

Tic Tac Toe is a solved game. With perfect play, the game always ends in a draw. The computer uses minimax algorithm to never lose.

Opening Moves

  • Center (position 5): Strongest first move. Controls 4 winning lines.
  • Corners (1,3,7,9): Second best. Each corner controls 3 lines.
  • Edges (2,4,6,8): Weakest. Only 2 winning lines each.

Fork Strategy

A fork creates two simultaneous threats to win. The opponent can only block one. Example: placing X at opposite corners forces a fork threat.

Blocking Forks

When opponent has two pieces in a line with an empty third, block immediately. To prevent forks, watch for patterns where opponent could create two threats on next move.

Winning Patterns

  • Any row, column, or diagonal of three identical marks.
  • Prioritize creating two-way forks.
  • If you have two in a row, complete the line.

Defensive Play

If opponent takes center, take a corner. If opponent takes corner, take center or opposite corner. This maintains defensive solidity.

Mathematical Minimax

The computer evaluates each move using:

V(s) = max( min( V(s') ) ) for alternating players.

Score = +10 (computer win), -10 (human win), 0 (draw). Alpha-beta pruning cuts irrelevant branches.

Perfect Play Guarantee

With optimal strategy, the computer will never lose. To win, exploit human mistakes — the computer punishes every error.
